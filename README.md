# Simple Student & Teacher OOP App

A simple Android application built in Java to demonstrate the four pillars of Object-Oriented Programming (OOP).

## Features
- **Abstraction**: Uses an abstract `Person` class.
- **Inheritance**: `Student` and `Teacher` classes extend `Person`.
- **Polymorphism**: Uses parent references to handle different child objects at runtime.
- **Encapsulation**: Private fields with public getters and setters in the `Student` class.

## Project Structure
- `Person.java`: Abstract parent class.
- `Student.java`: Subclass representing a student with marks.
- `Teacher.java`: Subclass representing a teacher.
- `MainActivity.java`: UI logic and OOP implementation demo.

## How to use
1. Enter a name.
2. Enter marks (only for Students).
3. Click "Student" or "Teacher" to see the OOP principles in action!
